One-Time Setup Guide
🧩
Do this once. After setup, you can call the skills naturally inside Claude whenever you need deeper thinking.
1. Choose where you want to use skills
Use Claude.ai for the easiest workflow, or Claude Code if you want skills available inside coding/project folders.
2. Open each skill URL
Use the links in the table above. For GitHub, open the SKILL.md file. For MCP Market, open the skill page and follow the install or copy instructions shown there.
3. Save each skill as its own folder or reusable instruction
Keep the skill name clear: premortem, steelman, red-team, first-principles. This makes it easier to call later.
4. Add the skill to Claude
In Claude.ai, add it as a Skill or reusable project instruction if your workspace supports Skills. In Claude Code, place the skill folder where Claude Code can reference project skills.
5. Test with one simple prompt
Ask Claude to use the skill on a small idea. If the output is too soft, add: “Be specific, harsh, and practical.”
Direct Workflow
✅
Use this exact workflow whenever you are building an idea, campaign, offer, script, landing page, or product.
1. Start with First Principles
Make Claude rebuild the idea from what is actually true.
2. Run Premortem
Ask Claude why this could fail 6 months from now.
3. Run Steelman
Ask Claude to argue the strongest opposite view.
4. Run Red Team
Ask Claude to attack your final plan like a brutal critic.
5. Rewrite with fixes
Ask Claude to turn the criticism into a stronger final version.
Universal workflow prompt:
I am working on [idea/campaign/offer/script/landing page].
First, break it down from first principles.
Then run a premortem and explain why it might fail 6 months from now.
Then steelman the opposite argument.
Then red team it brutally.
Finally, give me the improved version with specific fixes.
Ready-to-Copy Skill Prompts
Premortem
Use the Premortem skill on this idea. Imagine it failed 6 months from now. Work backwards and tell me the most likely reasons it failed, the early warning signs I ignored, and the fixes I should apply now.
Idea: [paste your idea]
Steelman
Use the Steelman skill. Argue the strongest possible case against my current belief or plan. Do not give weak objections. Make the opposite side sound intelligent, evidence-based, and hard to dismiss.
My plan/belief: [paste your plan]
Red Team
Use the Red Team skill. Attack this like a brutal critic, competitor, skeptical customer, and investor. Find unclear claims, weak proof, bad logic, conversion leaks, and trust issues. Then give me a fix list ranked by impact.
Asset: [paste pitch/offer/script/page]
First Principles
Use the First Principles skill. Break this down from what is actually true, not from trends, assumptions, or generic advice. Identify the core variables, constraints, incentives, and leverage points. Then rebuild the best solution from scratch.
Topic: [paste topic]
One Practical Example
🎯
Example use case: You are launching an AI ad creative service and want Claude to make the offer sharper before publishing it.
Example prompt:
I am launching an AI-powered ad creative service for brands that need high-quality Meta, TikTok, and Google ad creatives without hiring a full design team.
Use this workflow:
1. Break the offer down from first principles.
2. Run a premortem: why would this fail after 6 months?
3. Steelman the opposite view: why would a brand not buy this?
4. Red team the landing page angle, pricing, proof, and CTA.
5. Rewrite the final offer in a sharper, clearer, more conversion-focused way.
Be direct, specific, and practical.
